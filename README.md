# 1. Setup

1. Create venv
    ```bash
    python -m venv venv
    ```
2. Activate the venv
    ```bash
    venv\Scripts\activate
    ```
3. Install the requirements.txt
    ```bash
    pip install -r requirements.txt
    ```


# 2. Run

1. Activate Virtual Environment (If activate already, don't need to do this)
    ```bash
    venv\Scripts\activate
    ```

2. Run this command
    ```bash
    python main.py
    ```
